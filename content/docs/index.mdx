## Repositories

Source code is available at [github.com/colossus-credit](https://github.com/colossus-credit):

| Repository | Language | Description |
|-----------|----------|-------------|
| [emv-validator](https://github.com/colossus-credit/emv-validator) | Solidity | ERC-7579 Validator and Executor for ERC-4337 |
| [emv-card-sim](https://github.com/colossus-credit/emv-card-sim) | Java | ColossusNet JavaCard applet simulator |
| [emvpt](https://github.com/colossus-credit/emvpt) | Rust | Minimum Viable Payment Terminal |
| [kernel](https://github.com/colossus-credit/kernel) | Solidity | ZeroDev Kernel (ERC-4337 smart account) |
| [devnet](https://github.com/colossus-credit/devnet) | JavaScript | ColossusNet local development network |
| [rollup-boost](https://github.com/colossus-credit/rollup-boost) | Rust | Sidecar to enable rollup extensions |
| [builder-playground](https://github.com/colossus-credit/builder-playground) | Go | End-to-end block building environment |
