Processes an ISO 8583 ARQC (Authorization Request Cryptogram) message and returns an ARPC.

**Wire Format:** Binary ISO 8583 message (not JSON). See [`ARQC`](/api-reference/schemas#arqc) schema for logical structure.

**Required ISO 8583 Fields:**
- **DE55**: ICC/EMV Data (TLV encoded) - see [`EMVData`](/api-reference/schemas#emvdata) schema
- **DE62**: RSA public key + signature (TLV encoded) - see [`RSASignature`](/api-reference/schemas#rsasignature) schema

**Processing Flow:**
1. Parse ISO 8583 ARQC message
2. Extract EMV data from DE55
3. Extract RSA public key and signature from DE62
4. Look up account in database by public key
5. Build ERC-4337 UserOperation
6. Submit transaction to blockchain
7. Return ISO 8583 ARPC response (see [`ARPC`](/api-reference/schemas#arpc) schema)
